# اختبار قدرات محاكي

مشروع Vite/React عربي للتدريب اليومي على اختبار القدرات العامة. يعمل محليًا ويحفظ الحسابات، بنك الأسئلة، ودفتر الأخطاء داخل `localStorage`.

## التشغيل

```bash
npm install
npm run dev
```

## بنية المشروع

```text
index.html
package.json
README.md
src/
  App.jsx
  main.jsx
  questionTools.js
  styles.css
```

## شروط قبول السؤال

لا يقبل الموقع السؤال إلا إذا تحققت الشروط التالية:

- `id` موجود.
- `type` يساوي `quantitative` أو `verbal`.
- إذا لم يوجد `type` يمكن استخدام `section: "كمي"` أو `section: "لفظي"` كاحتياط.
- `question` غير فارغ.
- `choices` تحتوي 4 خيارات بالضبط.
- كل الخيارات غير فارغة.
- `correctAnswer` موجود داخل الخيارات بعد التحويل.
- `explanation` غير فارغ.
- `difficulty` يساوي `easy` أو `medium` أو `hard`.

يدعم الموقع تحويل الخيارات من كائن:

```json
{ "A": "...", "B": "...", "C": "...", "D": "..." }
```

إلى مصفوفة، ويدعم تحويل `correctAnswer: "A"` إلى نص الخيار الصحيح.

## الاختبارات

- اختبار كمي: يسحب `type = quantitative` فقط.
- اختبار لفظي: يسحب `type = verbal` فقط.
- تدريب مختلط: يستخدم النوعين.
- كل سؤال له مؤقت 90 ثانية، وإذا انتهى الوقت يحسب خطأ وسبب الخطأ الافتراضي `و = وقت`.

## دفتر الأخطاء

بعد الاختبار يمكن تحليل كل خطأ وتسجيل:

- سبب الخطأ.
- التصحيح بسطر واحد.
- عدد تكرار الخطأ.
- إعادة حل السؤال.
- حذف السؤال من دفتر الأخطاء.
